Roulette Directional Distance Predictor

Spins are assumed to alternate direction (CW, CCW, CW, CCW, ...). For each spin it records the wheel distance travelled in that spin's direction. To predict the next spin, it uses a Callan–Symanzik style directional distance model:
• Uses the last 3 distances in the upcoming direction,
• Computes drift β and curvature γ,
• Predicts S₄ = S₃ + a·β + b·γ,
• a,b grow as we accumulate more distances in that direction,
• A Martingale on stake per number, resetting to base after 11 consecutive losses.

This is the initial stake per number for both Live and Simulator. Martingale doubles it on each losing bet, and after 11 consecutive losing bets it resets to this base amount.
