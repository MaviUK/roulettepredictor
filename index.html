Roulette Pocket Bias Predictor

This version uses a more exact pocket-level Dirichlet model.
• Each pocket 0–36 gets its own posterior probability
• Prior: Dir(α, α, ..., α)
• Posterior: Dir(α+n₀, α+n₁, ..., α+n₃₆)
• Posterior mean for each pocket: (α+nᵢ) / Σ(α+nⱼ)
• Edge for each pocket: Eᵢ = p̂ᵢ × 36 − 1
• This script does not restrict betting by a confidence threshold; it always ranks all pockets and bets the top N pockets by posterior mean
• CSV optimiser objective: lowest worst losing streak, then fewest losing days, then highest hit rate, then best P&L
• Martingale doubles stake per number on each loss and resets after 11 consecutive losses.

Base = starting stake per number for Live, Simulator, and CSV runs. If you cover multiple pockets, total stake = stake per number × number of covered pockets.

Current model parameters: alpha=1.00, lookback=ALL, coverCount=6. Upload and optimise from CSV to tune for lower losing streaks.
